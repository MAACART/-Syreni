# -Syreni

Um trabalho de: Helena Toro, Marina Taddei, Vinicius Hiroshi, William Peçanha, Miguel Alonso, Silvia Laurentiz. (2022)

CAP/ECA/USP (2022)

Syreni é melhor visualizado no navegador chrome, em um desktop.



Motivado pela inquietude causada pela relação do ser humano com o oceano, Syreni entrega ao interator o controle sobre as forças da natureza.

Open Source: https://github.com/MAACART/-Syreni/
